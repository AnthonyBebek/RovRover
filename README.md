# RovRover PlatformIO Project

This project runs the RovRover driver deployed on a ESP32-S2 Mini board.

## Structure
- `src/` — Main source code
- `include/` — Header files
- `platformio.ini` — PlatformIO configuration

## Quick Start
1. Open this folder in VS Code with PlatformIO extension installed.
2. Select your board in `platformio.ini`.
3. Edit `src/main.cpp` as needed.
4. Build and upload using PlatformIO commands.

## Example
The default `main.cpp` prints messages to the serial monitor every second.

For more details, see [PlatformIO documentation](https://docs.platformio.org/).
